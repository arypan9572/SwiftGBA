RustyGBA — Game Boy Advance Emulator

Free GBA emulator running entirely in your browser via WebAssembly. No downloads, no installs, no tracking. Built with Rust for speed and safety. Load your ROM and play instantly.

Play GBA Games Online

RustyGBA is a free, open-source Game Boy Advance emulator that runs entirely in your browser. Built with Rust and compiled to WebAssembly, it delivers near-native performance with zero setup. Your ROMs never leave your device — everything is processed locally.

Features

  • Full GBA emulation — CPU, graphics, audio, and input
  • Save support — cartridge saves persist in your browser
  • Save states — quick save and load with F5/F8
  • Keyboard and gamepad support
  • 100% client-side — no uploads, no servers, no tracking
  • Built with Rust — zero unsafe code, maximum performance

Controls

Z = A, X = B, Arrow Keys = D-Pad, Enter = Start, Shift = Select, A = L, S = R
